# Calculator 🧮

This is a simple arithmetic calculator built using HTML, CSS, and JavaScript.

## 🚀 Features
- Addition ➕
- Subtraction ➖
- Multiplication ✖️
- Division ➗

## 💡 How to Use
1. Clone the repository
2. Open `index.html` in your browser
3. Start calculating!

Calculator/
│
├── index.html
├── style.css
└── script.js

## 🛠️ Tech Stack
